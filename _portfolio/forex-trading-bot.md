## Overview

A sophisticated **automated trading bot** for forex markets, combining Python and Java for rapid trade execution and comprehensive strategy testing.

## Key Features

- **Automated Trading**: Executes trades automatically based on predefined strategies using MetaTrader integration
- **Multi-Market Analysis**: Integrated real-time economic indicators from both Asia and Europe
- **Risk Management**: Dynamic stop-loss and take-profit mechanisms to protect capital
- **Backtesting Engine**: Extensive historical data analysis to validate and optimize trading strategies

## Technologies Used

- Python
- Java
- MetaTrader API
- TA-Lib (Technical Analysis Library)
- Risk Management Algorithms

## Technical Highlights

The bot processes real-time market data and economic indicators from multiple global markets, simulating diplomatic decision-making processes in financial technology.

Key risk management features include:

- Dynamic position sizing based on account equity
- Adaptive stop-loss levels based on market volatility
- Take-profit optimization using historical performance data
- Comprehensive logging and performance analytics
